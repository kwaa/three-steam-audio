## What is Three Steam Audio?

**Three Steam Audio** brings Valve's [Steam Audio](https://github.com/ValveSoftware/steam-audio) SDK to the web, with first-class support for both imperative Three.js code and declarative [React Three Fiber](https://r3f.docs.pmnd.rs/) scenes.

<CardGrid stagger>
  <Card title="HRTF spatialization" icon="rocket">
    Binaural rendering for headphones and accurate 3D positioning.
  </Card>
  <Card title="Direct path simulation" icon="seti:audio">
    Distance attenuation, air absorption, directivity, occlusion, and transmission.
  </Card>
  <Card title="Acoustic geometry" icon="codePen">
    Mark Three.js meshes as static or rigid dynamic acoustic geometry using built-in materials.
  </Card>
  <Card title="Parametric reflections" icon="seti:svg">
    Real-time reflection simulation with per-source reflections and listener reverb.
  </Card>
</CardGrid>
